# Medical-Agents-Hub
这是一个围绕“AI + 医疗”逐步构建的开源知识社区，系统汇聚人工智能与医疗健康交叉领域的各类资源，内容涵盖学习课程体系、技术与论文前沿、开源资源和工具、AI医疗治理等内容，共同服务于研究人员、医疗从业者、AI开发者与医疗教育工作者。

🗃️ 学习资源

从人工智能通识到医疗智能体进阶的学习资源，内容既包括人工智能基础、自然语言处理在临床文档中的应用，也包含医疗智能体的入门综述与典型架构解析，形式涵盖课件、视频讲解、智能体示例和实践作业，方便不同背景的朋友按需学习，逐步建立起完整的知识结构。

🗃️技术与论文

在技术与论文前沿方面，我们持续跟踪医疗模型及医疗与AI结合的最新研究成果，整理来自会议和期刊的重要论文，重点关注大语言模型在医疗场景中的应用、医疗智能体的推理与决策机制等方向，同时辅以综述与解读，帮助大家更快把握领域的发展脉络。

🗃️开源资源与工具

开源医疗智能体、开源医疗模型、开源数据集与开源工具也是社区的重要组成部分，我们希望持续汇集丰富的资源，帮助大家降低技术使用门槛，促进成果共享与协作创新。

🗃️AI医疗治理

社区会持续收录AI医疗治理、产业研究报告、行业白皮书以及实践分享，提供可参考的治理框架与最佳实践参考，鼓励技术创新与社会责任并行。

🙏本项目以开源、开放与协作为根本原则，致力于降低医疗AI领域的学习与参与门槛，促进跨学科学术与产业交流，推动人工智能在医疗中的负责任创新与广泛落地。

我们欢迎每一位成员参与内容的贡献与优化，推动资源的开放共享，通过透明、健康的社区治理机制，让这个知识社区持续生长。
